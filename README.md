**A clean, minimalist Hyprland dotfiles setup for Arch-based Linux distributions.
This setup includes an install script for Hyprland, Tofi, Dunst, Waybar, Kitty, and Hyprlock, along with my Laniakea wallpaper generator, which creates a unique wallpaper at every boot or whenever reloaded with Mod+W. It also comes with five Laniakea GTK and Kvantum themes.**

Install script:

<pre>git clone --depth=1 https://github.com/visnudeva/niri-laniakea.git ~/niri-laniakea
cd ~/niri-laniakea
chmod +x install.sh
./install.sh</pre>

Main keybindings:
     
    - Mod+SPACE tofi (app launcher)
    - Mod+E wlogout (logout menu)
    - Mod+B firefox (web browser)
    - Mod+F thunar (file manager)
    - Mod+T kitty (terminal)
    - Mod+I geany (IDE)
    - Mod+W generate new Laniakea wallpaper
